# AC Language Compiler

A multi-target compiler for the AC programming language that generates code for 11+ backends including Python, JavaScript, C, C++, Rust, Go, Java, HTML, V, x86-64 assembly, and native binaries.

**Key characteristics:**
- Beginner-friendly syntax with clean, readable code
- Write once, compile anywhere approach
- Uses `$...$` string delimiters with escape sequence support
- Supports functions, recursion, control flow (IF/ELSIF/OTHER, WHILST, FOR loops)
- Collections: lists, tuples, dictionaries
- Special `@` operator for multiplication
- Event listener and widget system for GUI applications
- Native binary compilation via x86-64 assembly generation

**Current status:** All 10 backends have complete feature parity with Python's advanced features including string escaping, case-insensitive booleans, null handling, and the @ operator translation.
