AC Language

AC is a multi-target compiled language. Write once, compile to 12 backends.

File Header

AC->BNY   ← native binary
AC->PY    ← Python
AC->JS    ← JavaScript
AC->C     ← C
AC->CPP   ← C++
AC->Java  ← Java
AC->RS    ← Rust
AC->GO    ← Go

Control Flow

    ...

    x += 1

    ...

Functions

Make greet func(name)

Try / Catch

try
    risky()
catch
    report err
    Term.display err
after
    cleanup()

Math Library

use ilib math
Term.display math.pi
Term.display math.sin(x)
Term.display math.sqrt(16.0)
