## Features Implemented Across ALL Backends:

### 1. Enhanced String Handling ✓
- **unwrapDollars()** with full escape sequence support
- Handles: `\"`, `\\`, `\n`, `\t`
- Preserves `$` inside strings
- Backend-specific quote styles maintained

### 2. Conditional Event Listener Emission ✓
- **usesEventListener()** helper function
- Only emits keybind code when actually used
- Clean output without spam

### 3. Case-Insensitive Boolean Translation ✓
**ALL backends now support:**
- `true`/`True`/`TRUE` → target language's `true`
- `false`/`False`/`FALSE` → target language's `false`

### 4. Null Handling ✓
**ALL backends now translate null:**
- **Python**: `null` → `None`
- **JavaScript/HTML**: `null` → `null`
- **C++**: `null` → `nullptr`
- **C**: `null` → `NULL`
- **Go**: `null` → `nil`
- **Rust**: `null` → `None`
- **V**: `null` → `none`
- **Java**: `null` → `null`
- **ASM**: `null` → `0` (numeric context)

### 5. @ Operator Translation ✓
**ALL backends translate `@` → `*`:**
- Works in expressions
- Works in assignments
- Works in global variables
- Works in conditions
